# База знаний туристических рекомендаций (55 правил)

## Структура правил

Каждое правило содержит следующие атрибуты:

| Атрибут | Описание | Допустимые значения |
|---------|----------|---------------------|
| `priority` | Приоритет срабатывания (чем выше, тем важнее) | 270–340 |
| `climate` | Предпочитаемый климат | `warm` (теплый), `cold` (холодный), `mild` (умеренный), `any` (любой) |
| `season` | Сезон поездки | `summer`, `winter`, `spring`, `autumn`, `any` |
| `travel_type` | Тип отдыха | `relax`, `active`, `mixed`, `culture`, `health`, `business`, `eco`, `education` |
| `budget_rub` | Бюджет в рублях | числовой диапазон или условие (`≥`, `≤`, `-`) |
| `trip_days` | Длительность в днях | число или диапазон |
| `companions` | Состав поездки | `solo`, `couple`, `family`, `friends`, `any` |
| `service_level` | Уровень сервиса | `economy`, `standard`, `premium`, `any` |
| `hobby` | Хобби / увлечение | `hiking`, `museum`, `food`, `dance`, `any` |
| `visa_mode` | Визовый режим | `visa_free_only`  `visa_ready`, `any` |
| `insurance` | Наличие страховки | `yes`, `no`, `any` |
